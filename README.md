# 🖥️ 실행

```src/main/java/chess.ChessMain.java```를 통해 프로그램이 실행

## 1) 체스 게임 init
```
시작 : start / 종료 : end
입력 > 
```
- start : 체스 게임이 시작된다.
- end : 체스 게임이 종료된다.
- 다른 값들을 입력하면 실행되지 않는다.
## 2) 체스 게임 시작
```

입력 > start
```

- 폰만 세팅된 채로 게임이 시작되고, 세팅된 정보를 출력한다.

## 3) 체스 게임 종료
```
시작 : start / 종료 : end
입력 > end
체스 게임 종료
```
